## Predictor

Analyze football statistics and predict the result of matches

### Used:

##### Front-end

- react
- react-router
- mobx
- webpack
- typescript
- jest
- enzyme
- istanbul
- css-in-js
- material ui
- prettier

##### Back-end

- nodejs
- express
- mongoDB (mongoose)

##### To start the project type in your CLI:

`npm start`

See demo [here](https://predict0r.herokuapp.com/)
